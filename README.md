# 📊 MIS Reporting Dashboard — Excel

## 🗂️ Project Overview
An end-to-end **MIS Reporting Dashboard** built in Microsoft Excel analyzing **300+ sales records** across 12 months, 5 regions and 5 product categories. Includes Monthly, Weekly and Regional performance reports with KPI tracking and data visualization.

---

## 🛠️ Tools Used
- 📗 Microsoft Excel (Advanced)
- 🔄 Pivot Tables
- 🔢 SUMIF, COUNTIF, IFERROR, INDEX-MATCH, RANK
- 📈 Charts — Line Chart, Bar Chart
- ✅ Data Validation
- 🎨 Conditional Formatting

---

## 📁 Dataset Details
- 📌 **Records:** 300+ sales entries
- 📅 **Period:** January 2024 — December 2024
- 🗺️ **Regions:** North, South, East, West, Central
- 🛒 **Product Categories:** Electronics, Clothing, Grocery, Home Appliances, Sports

---

## 📋 Dashboard Sheets

| 📄 Sheet | 📝 Description |
|---|---|
| 🗃️ Raw Data | 300+ rows of structured sales data |
| 📅 Monthly MIS Report | Month-wise Sales vs Target with MOM Growth % |
| 🗺️ Regional Analysis | Region-wise performance with RANK formula |
| 📊 KPI Dashboard | 6 KPI Cards + Line Chart + Bar Chart |
| 🛒 Product Analysis | Category-wise Sales Achievement % |
| 📆 Weekly MIS | Week-on-Week Sales tracking for Q1 2024 |

---

## 📌 Key KPIs Tracked
- 💰 Total Sales vs Target
- 🎯 Achievement %
- 📈 Month-on-Month (MOM) Growth %
- 📊 Week-on-Week (WOW) Growth %
- 🏆 Region-wise RANK
- 💹 Average Sale per Unit

---

## 🔢 Key Formulas Used
- **SUMIF** — Region / Month / Category wise aggregation
- **IFERROR** — Error handling in all calculations
- **RANK** — Region performance ranking
- **INDEX-MATCH** — Dynamic data lookup
- **MOM Growth** = `(Current Month - Previous Month) / Previous Month`

---

 # 💡 Business Insights
- 🏆 Identified **top performing region** using RANK formula
